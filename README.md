# Air Quality Forecasting using Prophet

## Overview
Time series forecasting project to predict Relative Humidity (RH) using the UCI Air Quality dataset and Facebook Prophet.

## Tech Stack
- Python
- Pandas
- NumPy
- Prophet
- Google Colab

## Steps Performed
- Data cleaning and missing value handling
- Datetime preprocessing
- Time series forecasting using Prophet
- Model evaluation using MAE and RMSE

## Results
- MAE: 13.03
- RMSE: 15.62

## Platform
Implemented and executed on Google Colab.
